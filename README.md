# OpenCV-Color-Recognition

## Project Description
This project uses OpenCV to detect and recognize colors in an image.

## Features
- Detects Red, Green, and Blue colors.
- Draws a bounding box around each detected object.
- Displays the detected color name.

## Tools Used
- Python
- OpenCV
- NumPy
- Visual Studio Code
- Anaconda

## How to Run
1. Install the required libraries:
   ```bash
   pip install opencv-python numpy
   ```
2. Run the project:
   ```bash
   python color_detection.py
   ```

## Test Image
The project was tested on an image containing three colored balls (Red, Green, and Blue).

## Output
The program identifies each colored object and displays its color name with a bounding box.
